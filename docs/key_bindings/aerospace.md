# Aerospace
## Modifier keys
- `Super` (Command/⌘) - Main modifier for all keybindings

## Layout management

| Keybind | Action |
|---------|--------|
| `Super+,` | Cycle layouts (tiles ↔ horizontal/vertical) |
| `Super+.` | Toggle fullscreen |
| `Super+/` | Toggle floating ↔ tiling layout |
| `Super+Shift+Up` | Join window with above |
| `Super+Shift+Down` | Join window with below |
| `Super+Shift+Left` | Join window with left |
| `Super+Shift+Right` | Join window with right |

## Window focus navigation

| Keybind | Action |
|---------|--------|
| `Super+Left` | Focus window to left |
| `Super+Right` | Focus window to right |
| `Super+Up` | Focus window above |
| `Super+Down` | Focus window below |

## Window moving

| Keybind | Action |
|---------|--------|
| `Ctrl+Super+Left` | Move window left |
| `Ctrl+Super+Right` | Move window right |
| `Ctrl+Super+Up` | Move window up |
| `Ctrl+Super+Down` | Move window down |

## Window resizing

| Keybind | Action |
|---------|--------|
| `Super+Shift+-` | Resize window smaller (smart) |
| `Super+Shift+=` | Resize window larger (smart) |

## Workspace management

| Keybind | Action |
|---------|--------|
| `Super+1..9,0` | Switch to workspace 1-10 |
| `Super+Shift+1..9,0` | Move window to workspace 1-10 (with focus) |
| `Super+Tab` | Switch to previous workspace (back-and-forth) |
| `Super+Shift+Tab` | Move workspace to next monitor (wrap around) |

## Application launching

| Keybind | Action |
|---------|--------|
| `Super+Return` | Open terminal (kitty + zellij) |
| `Super+E` | Open browser (primary) |
| `Super+Q` | Open browser (alternate) |
| `Super+M` | Open mail (primary) |
| `Super+K` | Open mail (alternate) |

## Binding modes

### Main mode
- Default mode for all window management and navigation

### Service mode
Enter with `Super+;`

| Keybind | Action |
|---------|--------|
| `Esc` | Reload config and exit mode |
| `R` | Flatten workspace tree (reset layout) |
| `F` | Toggle floating/tiling layout |
| `Backspace` | Close all windows except current |

### Apps mode
Enter with `Super+Esc`

| Keybind | Action |
|---------|--------|
| `Esc` | Reload config and exit mode |
