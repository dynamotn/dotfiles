# Kitty
## Modifier keys

- `Ctrl+Shift`: Main modifier

## Clipboard operations

| Keybind | Action |
|---------|--------|
| `Ctrl+Shift+C` | Copy to clipboard |
| `Ctrl+Shift+V` | Paste from clipboard |
| `Super+V` | Paste from clipboard (macOS) |
| `Ctrl+Shift+S` | Paste from selection |
| `Shift+Insert` | Paste from selection |
| `Ctrl+Shift+O` | Pass selection to program |

## Scrolling

| Keybind | Action |
|---------|--------|
| `Ctrl+Shift+Up` / `Ctrl+Shift+K` | Scroll line up |
| `Super+Up` | Scroll line up (macOS) |
| `Ctrl+Shift+Down` / `Ctrl+Shift+J` | Scroll line down |
| `Super+Down` | Scroll line down (macOS) |
| `Ctrl+Shift+Page Up` | Scroll page up |
| `Super+Page Up` | Scroll page up (macOS) |
| `Ctrl+Shift+Page Down` | Scroll page down |
| `Super+Page Down` | Scroll page down (macOS) |
| `Ctrl+Shift+Home` | Scroll to top |
| `Super+Home` | Scroll to top (macOS) |
| `Ctrl+Shift+End` | Scroll to bottom |
| `Super+End` | Scroll to bottom (macOS) |
| `Ctrl+Shift+Z` | Scroll to previous shell prompt |
| `Ctrl+Shift+X` | Scroll to next shell prompt |
| `Ctrl+Shift+H` | Show scrollback in pager |

## Window Management

| Keybind | Action |
|---------|--------|
|`Ctrl+Shift+Enter`|New window|
|`Ctrl+Shift+N`|New OS window|
|`Ctrl+Shift+W`|Close window|
|`Ctrl+Shift+]`|Next window|
|`Ctrl+Shift+[`|Previous window|
|`Ctrl+Shift+F`|Move window forward|
|`Ctrl+Shift+B`|Move window backward|
|`Ctrl+Shift+R`|Start resizing window|
|`Ctrl+Shift+1..0`|Focus specific window (1-10)|

## Tab management

| Keybind | Action |
|---------|--------|
|`Ctrl+Shift+Right`|Next tab|
|`Ctrl+Shift+Left`|Previous tab|
|`Ctrl+Shift+T`|New tab|
|`Ctrl+Shift+Q`|Close tab|
|`Ctrl+Shift+.`|Move tab forward|
|`Ctrl+Shift+,`|Move tab backward|
|`Ctrl+Shift+Alt+T`|Set tab title|

## Font

| Keybind | Action |
|---------|--------|
|`Ctrl+Shift+=` / `Ctrl+Shift++`|Increase font size|
|`Super+=`|Increase font size (macOS)|
|`Ctrl+Shift+-`|Decrease font size|
|`Super+-`|Decrease font size (macOS)|
|`Ctrl+Shift+Backspace`|Reset font size|
|`Super+0`|Reset font size (macOS)|

## Hints & text selection

| Keybind | Action |
|---------|--------|
|`Ctrl+Shift+E`|Open URL|
|`Ctrl+Shift+P F`|Select path and insert|
|`Ctrl+Shift+P L`|Select line and insert|
|`Ctrl+Shift+P W`|Select word and insert|
|`Ctrl+Shift+P H`|Select hash and insert|
|`Ctrl+Shift+P N`|Select linenum and open|
|`Ctrl+Shift+P Y`|Select hyperlink and open|
